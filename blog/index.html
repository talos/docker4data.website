what it is

Every open data set in the United States, in SQL.

Open data is great

open data portals are not

Existing open data portals are focused on views of single data sets in the browser.

This is great until you need to do deep analysis or build a business on it.

it doesn't need to be this hard

Downloading big datasets can take hours. It can take many more hours finding the location of important small datasets, especially those not living on the same portal. Add in a few more hours to load everything into a real database and figure out the relations, and you've spent days (or thousands of dollars) on something trivial.

there's a better way

Docker for data lets you plug directly into a geospatial-enabled (PostGIS) SQL database with thousands of US open data sets already loaded and ready to go.

Explore what's included

Every dataset hosted on a Socrata open data portal is accessible with SQL using Docker for Data. More to come.
